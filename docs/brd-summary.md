# BRD Summary

The PRDF Loan Management System digitizes the full loan lifecycle from client onboarding through repayment tracking. It provides two coordinated application surfaces: a client-facing portal for applicants and borrowers, and an internal admin dashboard for PRDF staff, approvers, and management.

The target operating outcomes are faster loan processing, stronger compliance, centralized document handling, clearer client communication, and better operational reporting. The platform is expected to support both self-service and staff-assisted onboarding patterns, with a UX simple enough for rural and peri-urban usage contexts.

MVP scope centers on:

- Client registration, profile capture, application submission, status tracking, and document upload
- Admin review workflows, approval or rejection handling, disbursement capture, repayment tracking, and reporting
- Notifications, role-based access control, and audit logging for major actions

The full requirements baseline is documented in [system-specification.md](./system-specification.md).
